Crud Market (REST API)
===============================================================================
A REST API for a blatantly fake E-Commerce application.

## Commands:

| Command       | Description                           |
| ------------- | ------------------------------------- |
| `npm start`   | Starts the server in production mode. |
| `npm dev`     | Starts the server in dev mode.        |
| `npm test`    | Run all unit tests.                   |
